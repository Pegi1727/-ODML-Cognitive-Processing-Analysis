The analysis was conducted using Linear Mixed-Effects Models (LMM) to account for the nested structure of the data (participants within groups). Key metrics analyzed include:
- Linguistic Accuracy (Correctness of output)
- Speech Onset Latencies (Cognitive processing speed)
- Long-term retention rates
